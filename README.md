#  Notes App

A simple and elegant Notes App built using **HTML, CSS, and JavaScript**.  
Users can add, edit, delete notes, and all notes are saved using **localStorage**, so they persist after page refresh.

---

##  Features

-  Add new notes
-  Edit existing notes
-  Delete notes
-  Data stored in browser using localStorage
-  Modern glassmorphism UI

---

##  Tech Stack

- HTML
- CSS
- JavaScript (DOM Manipulation & LocalStorage)
